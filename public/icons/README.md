# Icons Directory

This directory is for static icon files downloaded from Figma.

## Usage

- **SVG/Icons files**: Save them directly here
- Access them in your components using: `/icons/your-icon-name.svg`

Example:
```vue
<img src="/icons/logo.svg" alt="Logo" />
```

Or in CSS:
```css
background-image: url('/icons/icon.svg');
```
